## 心学

> 龙场悟道

```
道不远人，人之为道而远人，不可以为道，君子志于道；
造次必于是，颠沛必于是，身居逆境，尚能求道，近乎道矣
学为君子，天下归仁
---孔子
```

```
学问之道无他，求其放心而已矣，悟道由心而生，向心问道
君子所性，仁义礼智根于心
---孟子
```

```
一草一木皆有至理，如能豁然贯通，便能终知天理
豁然贯通，终知天理
---朱熹
```

```
人皆有是心，心皆具是理
人心至灵，此理至明，万物皆备于我矣
---陆九渊
```

```
心即是理，圣人之道，吾性自足，不假外求
---王阳明
```
