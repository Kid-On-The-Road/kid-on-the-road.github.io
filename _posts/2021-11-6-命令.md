## docker命令

- 查看容器在linux上的存储位置

```bash
docker inspect -f "{{.GraphDriver}}" 容器ID
```

- 删除所有镜像

```bash
docker rmi $(docker images -q)
```

- 启动所有容器

```bash
docker start $(docker ps -a -q)
```

- 关闭所有容器

```bash
docker stop $(docker ps -a -q)
```

- 将容器保存为镜像

```bash
docker commit 容器名 存储的镜像名
```

- 保存镜像为文件

```bash
docker save -o 存储的文件名.tar 镜像名
```

- 恢复镜像

```bash
docker load -i 镜像名.tar
```
